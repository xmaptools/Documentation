# Spider Module

*Based on XMapTools 4.5 embedded documentation – Help file version 03.12.2021*

---

This page describes the **Spider Module** which can be used to generate spider plots for LA-ICP-MS data.

## How to Generate Spider Plots

**Step 1:** Select the normalisation data from the *Normalisation* drop-down menu <img src="https://raw.githubusercontent.com/xmaptools/XMapTools_Public/main/Program/Dev/help/img/SS_LAICPMS_spider_norm.png" alt="Normalisation" width="20" style="display: inline; vertical-align: middle;">

**Step 2** (optional): Use the **Auto Contrast** <img src="https://raw.githubusercontent.com/xmaptools/XMapTools_Public/main/Program/Dev/help/img/icon_magic-wand.png" alt="Auto Contrast" width="20" style="display: inline; vertical-align: middle;"> button to automatically adjust the colour range.

**Step 3:** Select pixels to plot using one of the ROI types:
- <img src="https://raw.githubusercontent.com/xmaptools/XMapTools_Public/main/Program/Dev/help/img/icon_target.png" alt="Circle" width="20" style="display: inline; vertical-align: middle;"> **Circle**
- <img src="https://raw.githubusercontent.com/xmaptools/XMapTools_Public/main/Program/Dev/help/img/icon_placeholder.png" alt="ROI" width="20" style="display: inline; vertical-align: middle;"> **ROI**
- <img src="https://raw.githubusercontent.com/xmaptools/XMapTools_Public/main/Program/Dev/help/img/icon_map.png" alt="Path" width="20" style="display: inline; vertical-align: middle;"> **Path**

**Step 4** (optional): Activate the **Hold on** <img src="https://raw.githubusercontent.com/xmaptools/XMapTools_Public/main/Program/Dev/help/img/SS_spider_holdon.png" alt="Hold on" width="20" style="display: inline; vertical-align: middle;"> mode to select multiple regions of interest. Simply repeat Step 3 to add more selections (circle or ROI). It is possible to edit and move any selection and the plot will be updated.

::: tip
Uncertainties are plotted on the graph. It is possible to choose to display '1se' (1sd/√N) or '1sd' <img src="https://raw.githubusercontent.com/xmaptools/XMapTools_Public/main/Program/Dev/help/img/SS_spider_uncertainty.png" alt="Uncertainty" width="20" style="display: inline; vertical-align: middle;">.
:::
